# Mirror Glove Open Hardware

Mirror Glove is an open hardware wearable interface designed for finger motion capture using flexible resistive sensors, an ESP32-C6 Super Mini microcontroller, signal conditioning circuitry, firmware, and a labeled gesture dataset for assistive technology research.

The project was developed as an experimental platform for acquiring analog signals generated by finger flexion. The system uses five flexible sensors positioned on a glove, one for each finger, and converts finger movement into measurable electrical signals. These signals can be used for gesture analysis, motion mirroring, actuator control experiments, and preliminary gesture classification in external computational environments.

## Project goals

* Capture finger motion using five flexible resistive sensors.
* Condition analog signals using voltage dividers and passive RC low-pass filtering.
* Read sensor signals using an ESP32-C6 Super Mini.
* Organize sensor data into a structured gesture dataset.
* Support experimental motion mirroring using a transmitter and receiver architecture.
* Provide open hardware documentation for assistive technology, wearable interfaces, embedded systems, and prosthetics research.

## Repository structure

* `hardware/`: PCB design files, schematics, layout images, Gerber files, and hardware-related documentation.
* `firmware/`: ESP32 firmware for the glove transmitter and receiver/actuator unit.
* `docs/`: build instructions, calibration instructions, testing procedures, and operation notes.
* `bom/`: bill of materials with component specifications and quantities.
* `data/`: raw and processed datasets generated from the Mirror Glove.
* `analysis/`: scripts and notebooks used for signal analysis, PCA, machine learning models, and result generation.
* `figures/`: images, diagrams, plots, confusion matrices, and graphical outputs used in the manuscript.

## Hardware overview

The Mirror Glove prototype consists of a wearable glove with five flexible resistive sensors connected to a custom PCB. Each sensor is associated with one finger: thumb, index, middle, ring, and pinky. The analog signal from each sensor is conditioned using a voltage divider and passive RC low-pass filtering before being read by the ESP32 analog inputs.

The hardware was designed to support data acquisition, gesture recording, signal analysis, and future expansion for actuator control or wireless motion mirroring using ESP32-based communication.

## Software overview

The firmware is divided into two main modules:

1. Glove transmitter firmware: reads the five flexible sensors and organizes the sensor values.
2. Receiver/actuator firmware: receives sensor data and can convert proportional finger motion values into actuator commands.

Artificial intelligence and gesture classification are not embedded in the ESP32 firmware in the current version. Machine learning analyses are performed externally using the collected dataset.

## Dataset

The dataset contains labeled samples collected from the Mirror Glove sensors during gesture execution. Each sample includes time information, gesture labels, and the readings from the five flexible sensors. The dataset is intended for signal analysis, exploratory visualization, and preliminary supervised gesture classification.
